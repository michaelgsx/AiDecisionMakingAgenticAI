# Project skills (AiDecisionMakingAgenticAI)

Cursor Agent Skills for this repo. Each subdirectory contains a `SKILL.md` the agent reads when you invoke that workflow.

| Skill | Use when |
|-------|----------|
| [generate-unit-tests](generate-unit-tests/SKILL.md) | Adding or extending JUnit 5 tests under `backend/src/test` |
| [commit-and-push](commit-and-push/SKILL.md) | Committing changes and pushing to `origin` |
| [merge-to-main](merge-to-main/SKILL.md) | Merging `v1` into `main` and pushing both branches |

Default integration branch: **`v1`**. Production/doc branch: **`main`**.
